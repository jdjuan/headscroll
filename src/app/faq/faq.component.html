Frequently Asked Questions

How does it work?

In plain English, I took about 5000 pictures and managed to teach a machine what a tilted head was, and what it was not.

In other words, I created a model using Google's Teachable Machine, put it on an Angular application, and scrolled a website every time the model told me there was a tilted head.


What's on the roadmap?

The next step is release HeadScroll as a native app (the kind you install in your computer) so that you can scroll any website without restrictions and with a better user experience.

But just like I told my mom when I was born, keep your expectations down.


Is it open source?

Yes! The code is available here. Contributions and issues are always welcomed.


Why can't you display certain websites?

It's complicated.
I'm using an iframe to embed the websites and that already adds a lot of contraints. There's not even a reliable way to know if a website was displayed correctly other than user's feedback, so you can imagine.
That's why a native app would likely be the way to go.
