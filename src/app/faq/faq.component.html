Frequently Asked Questions

How does it work?

In plain English, we took about 5000 pictures and managed to teach a machine what a tilted head was, and what it was not. In other words, we created a model using Google's Teachable Machine, put it on an Angular application, and scrolled a website every time the model told us there was a tilted head.


What's on the roadmap?

The next step is release HeadScroll as a native app (the kind you install on your computer) so that you can scroll any website without restrictions and with a better user experience. But just like we told our moms when we were born, keep your expectations down.


Is it open source?

Yes! The code is available here. Contributions and issues are always welcomed.
